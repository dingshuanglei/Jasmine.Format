Jasmine.Format 演示

1. HtmlFormatHelper - HTML 模板格式化

Format - 位置参数格式化（自动HTML编码）

var userInput = "<script>alert('test')</script>";
HtmlFormatHelper.Format("用户输入: {0}", userInput)
输出: 用户输入: <script>alert('test')</script>

FormatTemplate - 命名参数格式化

HtmlFormatHelper.FormatTemplate("姓名: {name}, 年龄: {age}",
  new KeyValuePair("name", "张三"),
  new KeyValuePair("age", "25"))
输出: 姓名: 张三, 年龄: 25

FormatTemplate - Dictionary 参数

var dict = new Dictionary<string, string> { {"title", "产品标题"}, {"price", "¥99.00"} };
HtmlFormatHelper.FormatTemplate("商品: {title}, 价格: {price}", dict)
输出: 商品: 产品标题, 价格: ¥99.00

2. HtmlSpan - Span 元素构建

HtmlSpan 基础用法

new HtmlSpan("普通文本").ToHtml()
输出: 普通文本

HtmlSpan 带颜色

new HtmlSpan("红色文本", "#ff0000").ToHtml()
输出: 红色文本

HtmlSpan 带颜色和样式

new HtmlSpan("带样式文本", "#0066cc", "font-weight:bold; text-decoration:underline;").ToHtml()
输出: 带样式文本

3. HtmlP - 基础用法

Add - 添加普通文本

new HtmlP().Add("普通段落文本").ToHtml()

普通段落文本

Add + AddSpan - 链式调用（文本 + Span）

new HtmlP().Add("您好，").AddSpan("张三", "#0066cc").Add("，欢迎使用！").ToHtml()

您好，张三，欢迎使用 Jasmine.Format！

HtmlP(style) - 带样式的段落

new HtmlP("color:#666; font-size:14px;").Add("带样式的段落").ToHtml()

带样式的段落

AddRaw - 添加原始 HTML

new HtmlP().Add("文本前的空格: ").AddRaw("这是粗体文本").Add("，后面是普通文本").ToHtml()

文本前的空格: 这是粗体文本，后面是普通文本

AddSpan(HtmlSpan) - 添加预构建的 HtmlSpan

new HtmlP().AddSpan(new HtmlSpan("预构建的 HtmlSpan", "#993366")).ToHtml()

预构建的 HtmlSpan

4. HtmlP - 条件添加

AddSpanIf(bool) - 布尔条件判断

bool isAdmin = true;
new HtmlP().Add("用户状态：").AddSpanIf(isAdmin, "管理员", "#009900").AddSpanIf(!isAdmin, "普通用户", "#999999").ToHtml()

用户状态：管理员

AddIf(bool) - 条件添加文本

bool hasDiscount = false;
new HtmlP().Add("商品价格：¥99.00").AddIf(hasDiscount, " 限时优惠！", false).ToHtml()

商品价格：¥99.00

AddSpanIf<T>(source, predicate) - 基于对象条件判断

var user = new { Name = "王五", Score = 95 };
new HtmlP().Add("用户 ").AddSpan(user.Name, "#0066cc").Add(" 的等级：").AddSpanIf(user, u => u.Score >= 90, "优秀", "#009900").ToHtml()

用户 王五 的等级：优秀

AddRawIf(bool) - 条件添加原始 HTML

bool showWarning = true;
new HtmlP().Add("系统状态：正常").AddRawIf(showWarning, " <span style='color:#ff6600;'>⚠ 有新消息</span>").ToHtml()

系统状态：正常 ⚠ 有新消息

5. HtmlP - 批量添加

AddSpanRange(IEnumerable<string>) - 批量添加带样式的标签

var tags = new List<string> { "C#", "ASP.NET", "HTML", "CSS" };
new HtmlP().Add("技术标签：").AddSpanRange(tags, "#ff6600", "font-size:12px; padding:2px 6px; background:#fff3e0; border-radius:3px;").ToHtml()

技术标签：C#font-size:12px; padding:2px 6px; background:#fff3e0; border-radius:3px;ASP.NETfont-size:12px; padding:2px 6px; background:#fff3e0; border-radius:3px;HTMLfont-size:12px; padding:2px 6px; background:#fff3e0; border-radius:3px;CSS

AddRange(IEnumerable<string>) - 批量添加文本

var fruits = new List<string> { "苹果", "香蕉", "橙子" };
new HtmlP().Add("水果列表：").AddRange(fruits, "、").ToHtml()

水果列表：苹果、香蕉、橙子

AddSpanRange<T>(selector) - 使用选择器批量添加对象属性

var products = new List<Product> { ... };
new HtmlP().Add("商品列表：").AddSpanRange(products, p => $"{p.Name} (¥{p.Price})", "#333", "font-size:13px; margin-right:15px;").ToHtml()

商品列表：笔记本电脑 (¥5999)font-size:13px; margin-right:15px;无线鼠标 (¥99)font-size:13px; margin-right:15px;机械键盘 (¥299)

Add(text, count) - 重复添加文本

new HtmlP().Add("重复文本：").Add("★", 5, false).ToHtml()

重复文本：★★★★★

代码示例

var user = new { Name = "李四", Age = 30, Role = "高级工程师", Skills = new[] { ".NET", "React", "Docker" }, IsActive = true, Score = 98 };

var cardHtml = new HtmlP()
.Add("姓名：").AddSpan(user.Name, "#0066cc").Add("，")
.Add("年龄：").Add(user.Age.ToString()).Add("岁，")
.Add("角色：").AddSpan(user.Role, "#009900")
.AddIf(user.IsActive, "（在线）", false)
.ToHtml();

var skillTags = new HtmlP("margin-top:10px;")
.Add("技能：")
.AddSpanRange(user.Skills, "#666666", "display:inline-block; padding:3px 8px; background:#f0f0f0; border-radius:4px; font-size:12px; margin:2px;");
